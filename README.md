# Veterinaria Pro Dashboard

Sistema de gestión para clínica veterinaria.

## Tecnologías
- PHP
- MySQL
- Tailwind CSS
- Chart.js
- JavaScript Vanilla

## Páginas
- Dashboard
- Pacientes
- Veterinarios
- Citas Médicas

## Instalación
1. Clonar el repositorio
2. Importar `database.sql` en phpMyAdmin
3. Configurar `config/db.php`
4. Correr en XAMPP
